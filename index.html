About Startup Connect

Startup Connect is a discovery platform that aggregates live startup data, funding signals, and industry news to help founders, investors, and job-seekers find opportunities in the global ecosystem.

Built for an ALX assignment, it demonstrates real-world API integration, server-side deployment across multiple web nodes, and load-balanced traffic distribution.

APIs Used

  • NewsData.io — Startup & tech news headlines
    newsdata.io/docs
  • REST Countries — Country/region metadata
    restcountries.com
  • ExchangeRate-API — Currency context for funding figures
    exchangerate-api.com/docs
