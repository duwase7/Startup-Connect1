About Startup Connect

Startup Connect is a discovery platform I built for the ALX API assignment. It pulls live news from NewsData.io, uses REST Countries for country data, and ExchangeRate-API to show funding in local currencies.

The app is deployed across two Ubuntu web servers behind an Nginx load balancer, following the infrastructure setup we covered in the ALX programme.

APIs Used

  • NewsData.io - startup and tech news
    newsdata.io/docs
  • REST Countries - country and region info
    restcountries.com
  • ExchangeRate-API - currency conversion
    exchangerate-api.com
